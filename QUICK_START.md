# 🖱️ 自动鼠标控制应用 - 快速开始

## ⚡ 3 步启动

### 1️⃣ 安装依赖
```bash
cd D:\workspace2025\electron-vite-app
npm install
```

### 2️⃣ 启动应用
```bash
npm run dev
```

### 3️⃣ 应用自动打开！

---

## 📋 主要功能

✅ **移动鼠标** - 移动到指定位置  
✅ **点击操作** - 左键、右键、中键  
✅ **拖拽操作** - 从一点拖到另一点  
✅ **滚动操作** - 水平和垂直滚动  
✅ **屏幕选点** - 直接在屏幕上点击选择坐标  
✅ **定时执行** - 在指定时间自动运行  
✅ **延迟执行** - 可设置执行延迟  

---

## 🎯 创建第一个任务

1. 点击 **"➕ 创建新任务"**
2. 选择操作类型（如"移动鼠标"）
3. 点击 **"🎯 在屏幕上选择起始位置"**
4. 在屏幕上点击目标位置
5. 点击 **"创建任务"**
6. 点击任务列表中的 **"运行"** 按钮执行

## 🔧 常见问题

### Q: robotjs 安装失败（Windows）

**解决方案**：
1. 安装 Python 3（添加到 PATH）
2. 安装 Visual Studio Build Tools
3. 重新运行 `npm install`

### Q: 应用无法启动

**解决方案**：
```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### Q: 鼠标不工作

**检查**：
- 坐标是否在屏幕范围内
- 应用是否获得系统权限（macOS 需要）
- 按 F12 查看错误日志

### Q: 定时任务不执行

**检查**：
- 系统时间是否正确
- 定时时间是否已过期
- 任务是否还在应用中
